
---------------------------------Установка темы

1) Заходим на сайт генератор пустых ТЕМ http://underscores.me/
И генерируем там новую пустую тему

-Сохраняем архив с пустой темой на рабочий стол

2) Заходим в WP Appearance (Внешний вид) -> Themes -> Add new -> Upload Theme - и выбираем наш ZIP архив который мы скачали на рабочий стол с сайта генератора пустых темой

3) Переносим файлы версти в папку с нашей темой

 -В одном окне открываем папку с установленой темой в моем случаем это C:\xampp\htdocs\bogdan\WP_them\wp-content\themes\trivision
 - Стили от темы лежат в файле trivision/style.css

 -Далее открываем папку с нашей версткой и наченаем перемещать файлы ферстки в файлы темы WP
 -Далее копируем файл стилей от нашей верстки в папку с темой trivision (он должен иметь индивидуальное название в нашем случае default.css)
 -Далее копируем папку с картинками от верстки в нашем случае єто папка images - копируем ее в папку с темой trivision/
 -Далее файл index.php с разметкой пригодиться позже
 
 4) мы хотим изменить шаблон страници - открываем в папке с темой файл trivision/page.php 
 -Открываем файл page.php - И УДАЛЯЕМ ВСЕ ИЗ ЭТОГО ФАЙЛА
 - И теперь в пустой файл page.php мы будем помещать код ИЗ НАШЕГО СВЕРСТАНОГО index.php
 - Копируем весь код из index.php (нащ сверстаный файл) в файл page.php (в нашей теме) и СОХРАНЯЕМ
 
 5)Далее что-бы СТИЛИ B Джава файлы заработали ОТ НАШЕЙ сверстаной страници нужно 
  - И картинки что-бы подгружались надо так прописать (<img alt="Brand" src="<?php bloginfo("template_directory");?>/images/logo.png">)
 <?php bloginfo("template_directory");?>/    - этот код вставить во всех путях к ним
 (вставили в файле page.php в папке с темой)
 В нашем случаем выглядит так (       <link href="<?php bloginfo("template_directory");?>/default.css" rel="stylesheet"> )
 
 
 6) Далее что-бы тему можно было редактировать делаем следущее
 
  - Выводим заголовок страници  <?php the_title();?>  - пишем в месте где мы хотим что-бы WP выводил заголовок страници
  -Что0бы выводить КОНТЕНТ страници  нужно вставить нужное место 
		Там где выведем эти слова будет выводить контент WP
		<?php the_post(); ?>		- инициализирует данные
		<?php the_content(); ?>		- выводит данные
		
 7) Выводим имя сайта
 
	<?php bloginfo('name'); ?>
	<?php bloginfo('description'); ?>
	
 8) Меню сайта новой темы 
	 Вставляем код в сегмент меню
	 удаляем <ul><li> нашего старого меню и меняем на код снизу
	<?php wp_nav_menu('primary');?>
	
	-Что-бы добавить новую страницу Page->add new и потом нажимаем Published
	
	И меню автоматом появиться вторая страница ее название и будет работать ссылка на нее
	
 9) ПРоизвольные ПОЛЯ в теме
 
 <?php the_field(); ?> (нада установить плагин advanced custom fields)
 
 -Нада установить плагин - Нажимаем Plugins и в поиске пишем -> advanced custom fields - и устанавливаем плагин - И активировать его!
 
 И у нас в меню WP появляеться меню Custom Fields
 
 - Нажимаем на Custom Fields -> add new
 
 -потом называем поля именем нажимаем ->add_field
 ->Field Label -> name (например)
 ->Field Name  ->name
 ->Field Type ->image
 -> и ставим галочку на Image URL
 
 И НА ЖИМАЕМ Publish
 И потом в настройках Edit Field Group в Location -> Show this field group if ->выбираем  PAGE

И теперь если зайти в Pages и выбрать страницу которую будем редактировать - под редактором появилась КНОПКА Add Image
Нажимаем на кнопку и добавляем картинку которую мы хотим отобразить 
и в месте где она должна показываться добавляем 
<?php the_field("name"); ?>   - name - это поле  ->Field Name  ->name - которое мы указали при настройке добавления полей
