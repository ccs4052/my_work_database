API CURL Manual - https://developer.paypal.com/docs/api/#look-up-a-sale

1) Регистрируемся на https://www.paypal.com
2) Логинимся на https://www.paypal.com  заходим Profile -> Profile Setting -> My Selling tools -> Update -> View API Certificate
Создаем API User Name  нужны для создания аккаунта в SendBox на https://developer.paypal.com/

3)Dashboar -> My Apps & Credentials и создаем Create App -> после чего оно появляеться ниже и там будет нам clientID и secret для произведения оплаты

4) Заходим на https://developer.paypal.com/ -> DASHBOARD -> Sandbox -> Accounts и нажимаем Create Account

5) Затем нажимаем на выбраный аккаунт + Profile -> API Credentials и для дальнейшей работы нам понадобиться Username + Signature

6) Первый запрос мы делам что-бы получить секретный Token для дальнейшей оплаты

мануал https://developer.paypal.com/docs/integration/web/web-checkout/
