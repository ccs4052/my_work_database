 ---------- Дополнительные операторы
 > - больше 
 < - меньше
 = - присвоение
 == - сравнение по значению
 === - оператор строгого значения по значению и типу
 <= - ментше или равно
 != не равно
 <> - не равно ( синоним !=)
 && - И (действие 1 && действие 2 выполнялись ( и то и то))
 || - ИЛИ
 & - знак склеивания переменных ( например за функцией и вне функции)
 
 @ - если перед переменной установить этот символ то ошибка связаная с этой переменной в данной строке не выведеться

 ОПЕРАТОРЫ СРАВНЕНИЯ (Boolian или true или False)
$a == $b //TRUE если $a равно $b
$a === $b // TRUE если $a равно $b и имеет тот же тип
$a != $b // TRUE если $a не равно $b
$a !== $b // TRUE если $a не равно $b или в случае, если они разные типов
$a < $b
$a > $b
$a <= $b
$a >= $b
$a and $b // TRUE  если и $a и $b TRUE
$a or $b // TRUE если или $a или $b TRUE
!$a // TRUE если $a не TRUE
$a && $b // TRUE если $a и $b TRUE
$a || $b // TRUE если или $a или $b TRUE
$a and $b and $c
$a and $b or $c
$a and ($b and $c) - сначала выполняеться действие в скобках
